# Мини-игра про пакмана
Этот код - реализация популярной игры пакман. Функционал игры идентичен оригиналу. Вам предстоит управлять пакманом, избегать призраков, набирать очки и открывать новые уровни(пока в разработке).
Логика игры написана на JS, используется также CSS и HTML Canvas.

## Инструкция по установке:
1. Клонируйте репозиторий с помощью команды <code>git clone</code>
```shell
git clone https://github.com/powder-rose/Pacman-Game.git
```
2. Запустите приложение с помощью <b>live server</b> или аналогичной программы
3. Приятной игры! ❤️
## Управление:

- ***Вперед*** - W или ↑
- ***Назад*** - S или ↓
- ***Вправо*** - D или →
- ***Влево*** - A или ←

Начать новую игру - нажмите любую клавишу
